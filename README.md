# prototype-lab

A sandbox for quick prototypes, small tools, and proof-of-concept explorations — focused on delightful UX for families and kids.

---

## Projects

### 1. 🧸 Wunder — AI Toy Recommendations
> *Find the toy made for them.*

An AI-powered toy recommendation web app for parents of children ages 0–5. Enter your child's age, interests, and personality — Wunder uses Claude to return personalized, developmentally-informed toy picks with custom watercolor-style illustrations generated by QuiverAI.

| | |
|---|---|
| **Stack** | Node.js · Express · Claude API · QuiverAI SVG API |
| **AI** | Claude (`claude-sonnet-4`) for ranked recommendations + blurbs grounded in Lovevery/NAEYC developmental research |
| **Illustrations** | 40 pre-generated watercolor SVGs via QuiverAI, served statically |
| **Features** | Age slider · interest chips · budget filter · wishlist (localStorage) · client-side sort/filter · illustrated background |
| **Run** | `cd wunder && npm install && npm start` → http://localhost:3000 |

```
wunder/
├── server.js              # Express server + Claude API endpoint
├── data/toys.json         # 41 curated toys across 7 categories
├── scripts/
│   ├── generate-illustrations.js   # Pre-generates SVGs via QuiverAI
│   └── generate-background.js      # Generates page background SVG
└── public/
    ├── index.html
    ├── styles.css
    ├── app.js
    └── illustrations/     # Pre-generated watercolor SVGs
```

### 2. 🌸 Kids Memory Cards — Childhood Memory Keeper

A static web app to capture and preserve your child's precious moments and funny sayings. No server or setup needed — runs entirely in the browser.

| | |
|---|---|
| **Stack** | HTML · CSS · Vanilla JavaScript |
| **Storage** | Browser `localStorage` — no backend required |
| **Features** | Add/edit/delete memories · date tracking · export & import JSON backup · animated card UI |
| **Run** | Open `kids-memory-cards/index.html` directly in a browser |

```
kids-memory-cards/
├── index.html
├── styles.css
├── script.js
└── images/        # Character illustrations
```

## Getting Started

```bash
git clone <repo-url>
cd prototype-lab

# Run Wunder
cd wunder
cp .env.example .env   # add your ANTHROPIC_API_KEY and QUIVERAI_API_KEY
npm install
npm start
```
